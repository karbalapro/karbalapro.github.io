# muharram.github.io

This is a [Next.js](https://nextjs.org) project for the Interactive Ziyarat & Ashura Encyclopedia.

## Features
- Complete Ziyarat Ashura & Ziyarat Warith with translations.
- Encyclopedia of 274+ Karbala personas with biographies.
- Multi-language support (English, Persian, Arabic).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
